## 💭 什么是编程？

想象一下，你在教一个机器人做事：
- 你需要用机器人能听懂的语言
- 你的指令要非常清楚，一步一步来
- 编程就是用计算机能理解的语言，告诉它要做什么

Python就是一种编程语言，就像是和计算机沟通的语言！

## 📝 第一个程序：Hello World

在Python中，最简单的程序就是让电脑说话：

```python
print("你好，世界！")
```

**运行后会看到：**
```
你好，世界！
```

### 🔍 print是什么？

- `print()` 是Python的一个"命令"
- 括号里的内容会显示在屏幕上
- 就像是让电脑"打印"出你给它的文字

### ✏️ 试试看！

你可以让电脑打印任何内容：

```python
print("我叫小明")
print("今年18岁")
print("喜欢编程")
```

**输出：**
```
我叫小明
今年18岁
喜欢编程
```

3. 让电脑打印一行星星

## 💡 重要概念

1. **代码**：你写的指令
2. **运行**：让电脑执行你的指令
3. **输出**：电脑执行后显示的结果

## 🤔 思考题

为什么需要用 `"` 引号把文字包围起来？

- 因为这样Python才知道哪些是文字（要打印的内容）
- 不然Python会以为你在说别的命令

## 📚 项目中的实际应用

在咱们的项目中，有这样的代码：

```python
print("=" * 50)
print("名侦探作业帮 - 正在启动")
print("=" * 50)
```

这会在程序启动时显示一个漂亮的标题！
