# 成长日记功能规划方案

## 项目概述

### 背景
"名侦探作业帮"是一个以柯南侦探主题为核心的AI学习平台。现需要新增"成长日记"模块，让学生不仅能解决学业问题，还能记录成长、获得情感支持、培养自我管理能力。

### 核心理念
> "真相只有一个，成长也是如此" —— 每一天的记录都是解开自己成长密码的线索

### 目标用户
- 初高中学生（13-18岁）
- 有学业压力、需要情感出口
- 希望自我提升、培养好习惯

---

## 功能模块设计

### 模块一：日记记录 (核心功能)

#### 1.1 基础日记
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 文字记录 | 大文本框，支持记录每日感受、事件 | P0 |
| AI暖心回复 | 保存后立即获得AI的共情回应 | P0 |
| 历史查看 | 按时间倒序浏览所有日记 | P0 |

#### 1.2 日记增强功能
| 功能 | 描述 | 优先级 |
|------|------|--------|
| 日记模板 | 提供"学习反思"、"情绪宣泄"、"目标复盘"等模板 | P1 |
| 语音输入 | 支持语音转文字，方便快速记录 | P2 |
| 图片附件 | 可上传今日照片（如学习笔记、风景） | P2 |
| 隐私锁 | 部分日记可设为"仅自己可见"（加密） | P1 |

#### 1.3 AI回复风格
根据主题匹配不同的AI角色：

| 场景 | AI角色 | 回复风格 |
|------|--------|----------|
| 学业压力 | 小柯（柯南） | 理性分析、鼓励思考 |
| 情绪低落 | 小兰 | 温柔安慰、情感共鸣 |
| 分享喜悦 | 少年侦探团 | 欢快祝贺、一起开心 |
| 目标规划 | 阿笠博士 | 智慧建议、长远视角 |

## AI Prompt 设计要点

### 日记回复Prompt原则
1. **共情优先**：先认同用户的感受
2. **简短有力**：100-150字，不啰嗦
3. **积极引导**：给出正向的思考角度
4. **避免说教**：朋友语气，不是老师
5. **个性化**：根据用户历史调整风格

### 示例Prompt
```
你是"小柯"，一个温暖、有同理心的AI伙伴。
你的特点：
- 像朋友一样聊天，不说教
- 善于发现用户话语中的亮点
- 在用户低落时给予安慰，在用户开心时一起庆祝
- 偶尔用柯南的经典台词点缀（如"真相只有一个"）

用户刚写下这篇日记，请给出简短回应（100-150字）：
{diary_content}
```

## 总结

成长日记功能将"名侦探作业帮"从单纯的学业工具升级为**全方位成长伙伴**：

1. **情感出口**：学生可以倾诉压力、分享喜悦
2. **自我认知**：通过数据可视化了解自己的状态
3. **习惯培养**：打卡和计划系统帮助养成好习惯
4. **正向激励**：成就系统给予持续的正反馈

核心差异化：**AI不只是工具，而是陪伴成长的朋友**
